# 🎨 Background Color Changer

A simple web application built using **HTML, CSS, and JavaScript** that changes the webpage background color when different color buttons are clicked. This project demonstrates the use of **DOM manipulation** and **event handling** in JavaScript.

## 🚀 Features

- Change the background color with a single click.
- Implemented **5 color buttons** using JavaScript event listeners.
- Dynamic DOM manipulation.
- Clean and responsive user interface.
- Beginner-friendly project for understanding JavaScript events.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)


## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/background-color-changer.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

## 🎯 How It Works

- Click any of the five color buttons.
- The background color of the webpage changes instantly.
- JavaScript uses **event listeners** and **DOM manipulation** to update the background color dynamically.
